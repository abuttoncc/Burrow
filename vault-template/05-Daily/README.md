# 05-Daily (optional)

Daily notes, if you keep them (`YYYY-MM-DD.md`). The dashboard's today-list reads the current day's
file when present and degrades gracefully when absent. This layer is yours — the Burrow never
writes here except when you ask a routine to log something back to today.

Works well with the full PARA method (01-Projects / 02-Areas / 03-Resources / 04-Archive) if that's
your style — none of it is required by the Burrow discipline.
